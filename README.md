# Customer Shopping Behavior Analysis

## Project Overview

This project simulates a **corporate-level end-to-end data analytics workflow**, demonstrating how raw data can be transformed into **actionable business insights**.

It showcases the complete analytics lifecycle — from data preparation to business intelligence — using industry-standard tools.

### Workflow Breakdown

✅ **Data Preparation, Modeling & Exploratory Data Analysis (Python)**

* Cleaned and transformed raw data
* Performed exploratory data analysis (EDA)
* Prepared structured data for further analysis

✅ **Data Analysis (SQL)**

* Simulated real-world business scenarios
* Wrote queries to analyze:

  * Customer segments
  * Customer loyalty
  * Purchase behavior & key drivers

✅ **Visualization & Insights (Power BI)**

* Built an interactive dashboard
* Highlighted trends and patterns
* Enabled data-driven decision-making


## How to Use This Project

Follow these steps to run the project end-to-end:

### 1️⃣ Open Python Notebook

Open:

Customer_Shopping_Behavior_Analysis.ipynb

This notebook includes:

* Data import
* Data exploration (EDA)
* Data cleaning & preprocessing
* Feature engineering
* Connection setup for SQL database


### 2️⃣ Load Data into SQL Database

* Create a database in:

  * MySQL / PostgreSQL / SQL Server
* Run the Python notebook to:

  * Establish connection
  * Load cleaned data into SQL tables


### 3️⃣ Run SQL Analysis

Open:

customer_behavior_sql_queries.sql

Use this file to:

* Answer key business questions
* Analyze:

  * Revenue patterns
  * Customer segmentation
  * Product performance
  * Subscription impact

---

### 4️⃣ Build & Explore Dashboard (Power BI)

* Connect Power BI to your SQL database
* Open:
  
customer_behavior_dashboard.pbix

Inside the dashboard:

* Explore interactive visuals
* Apply filters (category, gender, shipping, etc.)
* Analyze trends and insights
